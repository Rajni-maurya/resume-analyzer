# 🤖 Resume Analyzer (Flask Project)

A web application that analyzes resumes and compares them with job descriptions to calculate a match score.

## 🚀 Features

- Upload resume (PDF/DOCX)
- Match Score Calculation
- Matched Skills Display
- Missing Skills Detection
- Attractive UI with Progress Bar

## 🛠️ Tech Stack

- Python (Flask)
- HTML, CSS, Bootstrap
- JSON

## ▶️ How to Run

1. Clone repo:
git clone https://github.com/your-username/resume-analyzer-ai.git

2. Install dependencies:
pip install -r requirements.txt

3. Run:
python app.py
